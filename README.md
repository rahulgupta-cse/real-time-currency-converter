This project is a Python-based Currency Converter that allows users to convert Indian Rupees (INR) into multiple international currencies such as USD, CNY, JPY, AED, and more.

It fetches real-time exchange rates using an API to ensure accurate conversions.

Features:
- Convert INR to 12+ global currencies
- Real-time exchange rate fetching using API
- Simple CLI (Command Line Interface)
- User-friendly menu system
- Error handling for invalid input

Technologies Used:
- Python
- Requests Library
- ExchangeRate API
